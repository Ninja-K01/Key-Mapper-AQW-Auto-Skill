# Key-Mapper-AQW-Auto-Skill
Key Mapper for AQW Pocket & Puffin Browser – Auto Skill Tool

A lightweight key-mapping utility designed to streamline gameplay in AdventureQuest Worlds (AQW) when played through mobile or browser environments such as AQW Pocket and Puffin Browser. It allows players to assign quick-access hotkeys for in-game skills, helping improve responsiveness and reduce manual input delays during combat.

This tool focuses on simplifying skill execution by organizing key bindings into customizable combos, making gameplay smoother and more efficient across supported platforms. It is designed for ease of use, fast setup, and flexible configuration based on different character skill rotations.

Key Features:

Custom key mapping for AQW skills
Supports combo-based skill sequences
Optimized for mobile and browser gameplay (Puffin / AQW Pocket)
Simple configuration for different class rotations
Lightweight and easy to run alongside the game

Designed to enhance convenience and control during gameplay by centralizing skill inputs into a faster, more accessible system.
